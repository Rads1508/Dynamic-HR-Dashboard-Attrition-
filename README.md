# Dynamic-HR-Dashboard-Attrition-
Excel Project for HR

It's an Excel Dashboard for HR based on Attrition of employees based on several factors.

It includes:
-Raw Data
-Transformed Data
-Dashboard

Things learnt:
-Cleaning of Data
-Formatting of Data Tables
-Employees KPIs
-Charts
-Custom Charts
-Dynamic Slicers
-DAX
-Power Query
-Pivot Tables
-Dynamic Legends etc..
